DayZ-Fov-Changer
================

Automated macro for DayZ users to change their fov for:
F2= A better shooting/sniping (FOV 0)
F1= Normal/default FOV
Blocks your mouse inputs during the macro, you can still run around.

You can change the code to the hotkeys you want. 
Open it with notepad and change "f1::" or "f2::", to any keybind you would like.

All Keybinds available are in here:
http://www.autohotkey.com/docs/commands/Send.htm

Choose one you like, and replace f1:: or f2:: with it. 
Remember to take off {}.
